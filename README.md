# TimeBookingAPI

### ***Запуск:***

```
make
```

### ***Просмотр логов***:

```
make logs
```

### ***Остановка***:

```
make down
```

### ***Сборка***:

```
make build
```

### ***Запуск тестов***:

```
make tests
```

### ***Swagger***:
http://localhost:8080/swagger/index.html
